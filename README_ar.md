# دُرى موهوبة الرياضيات

تطبيق ويب عربي بالكامل للرياضيات باسم **دُرى موهوبة الرياضيات**.
- يدعم **سؤالًا صوتيًا** و/أو **كتابيًا**.
- يدعم **إجابة صوتية** و/أو **مرئية (نصيّة مع صيغ رياضية LaTeX عبر MathJax)**.
- جميع الكلمات والأحرف والأرقام **بالعربية** مع تحويل الأرقام تلقائيًا إلى (٠١٢٣٤٥٦٧٨٩).
- متصل بـ **GPT** عبر خادم Node.js (ضع مفتاح OpenAI لديك).
- أصوات وتشجيع وحركات لطيفة في الواجهة.

> ملاحظة: الصوت بصوت «بنت صغيرة» يُحاكى عبر **متصفحك** باستخدام خاصية تحويل النص إلى كلام (SpeechSynthesis) باختيار صوت عربي ورفع طبقة الصوت (pitch). قد يختلف الصوت حسب النظام والمتصفح.

## المتطلبات
- Node.js ١٨+
- مفتاح OpenAI: احصل عليه ثم ضعه في `.env`

## التشغيل المحلي
١) فك الضغط ثم افتح المجلد في الطرفية:
```bash
cd durra-math-app
cp .env.example .env
# افتح .env وعدّل OPENAI_API_KEY إلى مفتاحك
npm install
npm run start
```
٢) افتح المتصفح على:
```
http://localhost:3000
```

## خصائص
- تحويل جميع الأرقام إلى صيغة عربية-هندية تلقائيًا.
- تعليمات صارمة للنموذج للإجابة **باللغة العربية فقط** والالتزام بالرياضيات.
- واجهة RTL، وأزرار للتحكم: سؤال صوتي/كتابي، وإجابة صوتية/مرئية.
- عرض الصيغ بمعيار LaTeX عبر MathJax.
- مؤثرات صوتية وحركات خفيفة تشجيعية.

## تخصيص
- يمكنك تغيير اسم التطبيق في `index.html`.
- نموذج الدردشة الافتراضي في `server.js` عبر المتغير `OPENAI_CHAT_MODEL`.
- إضافة أصوات أو صور في مجلد `public` كما تشاء.
